도경의 첫 번째 프로젝트

김도경이 IT프로젝트의 첫 시작으로 HTML을 선택하다.

  • HTML이란?

    • 인터넷 서비스의 하나인 월드 와이드 웹을 통해 볼 수 있는 문서를 만들 때 사용하는 웹 언어의 한 종류이다. 특히 하이퍼텍스트를 작성하기 위해 개발되었으며, 인터넷에서 웹을 통해 접근되는 대부분의 웹 페이지들은 HTML로 작성된다.
    HTML [Hypertext Markup Language]

    - 모든 태그에는 <> 이 표시가 무조건 들어가야하고, 마무리엔 < / >로 끝나야한다.

    처음으로 배운 tag(태그)는 h1~h6, strong 이라는 것을 배웠다.
    1. < h1~h6 >는 약자는 잘 모르겠지만 대충 heading 이라는 표현 같다. 즉, 글씨를 키우는 역할을 한다.
    2. < strong >은 글씨를 진하게 하는 역할을 한다.
    3. < br > 이라는 태그는 '줄바꿈' 태그인데, 줄을 바꾸는 의미만 있기 떄문에 끝에 < / >가 들어가지 않는다.
    4. < p > 라는 태그는 '단락' 을 의미한다. paragraph
    5. < img > 태그는 뒤에 src(source)의 약자를 넣어준다. 그런데 이미지 파일이 있어야한다.
    width는 폭, 너비 라는 뜻으로 사진 크기를 조절 가능하다.

    6. 부모자식 태그 < li > < /li > ul(Unordered List)- 연관된 부분 나눌 , ol(Ordered List) - 자동으로 숫자 기입

    가장 많이 쓰이는 태그★
    - < title >,< /title > 책으로 치면 표지와 같은 정보로써 사용
    - < meta char(문자)set(규칙)="~~" > 문자는 0과 1로 이루어져 있어서 파일 깨지지 않게 ~~
    - < body >,< /body > 약속임. 본문 설명 같은거~?
    - < head >,< /head >
    - < html >,< /html >
    - < a >.< /a > 하이퍼텍스트 링크 거는거 ex) < a href(참조해라는 뜻)="주소" target="_blank(이거는 새창으로 열기 )" title="html5 specification"(이거는 링크에 마우스 두면 설명 뜨게 하는거) >
